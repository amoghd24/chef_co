Chef Co API Documentation

Welcome to the Chef Co API documentation. This API allows you to manage menus, courses, menu items, and predict food quantities based on party size.

API Documentation

Explore our API using the interactive documentation interfaces:

Swagger UI ReDoc

Authentication

Most API endpoints require authentication. You can get a token by sending your username and password to:

/api-token-auth/

Then include the token in your requests with the following header:

Authorization: Token YOUR_TOKEN_HERE

Available Endpoints

The API provides endpoints for:

  • Menus - Create and manage menus
  • Courses - Manage course sections within menus
  • Menu Items - Food items within courses
  • Quantity References - Reference quantities for menu items
  • Party Orders - Orders for specific party sizes
  • Predictions - Generate and retrieve food quantity predictions

Prediction Workflow

To generate quantity predictions:

  1. Create a party order with POST /api/party-orders/
  2. Generate a prediction with POST /api/party-orders/{id}/predict_quantities/
  3. View past predictions with GET /api/predicted_quantities/
  4. View a specific prediction with GET /api/predicted_quantities/{id}/
